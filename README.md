<h1 align="center">E-Commerce Price Prediction</h1>
<p>The E-Commerce Price Prediction System is a web application that predicts product prices based on historical data and product features collected from major e-commerce platforms. The system helps users analyze market trends and estimate optimal product pricing using machine learning algorithms.</p>
<img width="1893" height="953" alt="Screenshot 2026-02-27 112001" src="https://github.com/user-attachments/assets/fc832f39-1668-438e-a6c9-c90006eebfc8" />
<p>The application collects product data such as title, brand, category, ratings, reviews, and previous prices. After data preprocessing and feature engineering, a machine learning model is trained to predict the expected price of a product.</p>
